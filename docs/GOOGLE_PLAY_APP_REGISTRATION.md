# Google Play Console 앱 등록 가이드

API 액세스 페이지에 접근하려면 먼저 앱을 등록해야 할 수 있습니다.

## 🎯 왜 앱 등록이 필요한가?

Google Play Console에서 API 액세스 기능을 사용하려면:
- 최소 1개 이상의 앱이 등록되어 있어야 합니다
- 앱이 없으면 API 액세스 페이지가 비활성화되거나 접근이 제한될 수 있습니다

**좋은 소식**: 앱을 실제로 배포할 필요는 없습니다! 등록만 하면 됩니다.

## 📱 앱 등록 단계

### 1단계: 앱 만들기

1. [Google Play Console](https://play.google.com/console)에 로그인
2. 메인 페이지에서 **앱 만들기** 또는 **Create app** 버튼 클릭
   - 또는 왼쪽 사이드바에서 **앱** → **앱 만들기**

### 2단계: 앱 정보 입력

다음 정보를 입력하세요:

#### 필수 정보

1. **앱 이름**: 
   - 예: "Taba"
   - 나중에 변경 가능

2. **기본 언어**: 
   - 한국어 선택 (또는 원하는 언어)

3. **앱 또는 게임**: 
   - **앱** 선택

4. **무료 또는 유료**: 
   - **무료** 선택 (나중에 변경 가능)

5. **개인정보처리방침 URL** (선택사항):
   - 나중에 추가 가능

### 3단계: 앱 만들기 완료

1. **앱 만들기** 또는 **Create app** 버튼 클릭
2. 앱이 생성되면 앱 대시보드로 이동

### 4단계: API 액세스 페이지 접근

앱이 등록된 후:

1. 왼쪽 사이드바에서 **설정** 클릭
2. **API 액세스** 탭 찾기
3. 또는 직접 URL 접근:
   ```
   https://play.google.com/console/u/0/developers/[YOUR_DEVELOPER_ID]/api-access
   ```

## ✅ 확인 사항

앱 등록 후 다음을 확인하세요:

- [ ] 앱이 Google Play Console에 등록되었는가?
- [ ] 앱 대시보드에 접근할 수 있는가?
- [ ] API 액세스 페이지에 접근할 수 있는가?

## 📝 참고 사항

### 앱을 실제로 배포할 필요 없음

- 앱을 등록만 하면 API 액세스 기능을 사용할 수 있습니다
- 실제로 Google Play Store에 배포할 필요는 없습니다
- 내부 테스트, 알파, 베타, 프로덕션 중 어떤 단계도 완료할 필요 없습니다

### 나중에 앱 정보 수정 가능

- 앱 이름, 설명, 아이콘 등은 나중에 언제든지 수정할 수 있습니다
- 지금은 최소한의 정보만 입력해도 됩니다

### 앱 삭제 가능

- 나중에 앱을 삭제할 수도 있습니다
- 하지만 API 액세스는 앱이 등록되어 있는 동안만 사용 가능합니다

## 🆘 문제 해결

### "앱 만들기" 버튼이 보이지 않아요

- 개발자 계정이 완전히 설정되었는지 확인
- 결제 프로필이 설정되어 있는지 확인 (일부 경우 필요)

### 앱을 등록했는데도 API 액세스가 안 돼요

- 몇 분 기다린 후 다시 시도 (시스템 반영 시간)
- 브라우저 캐시 삭제 후 재시도
- 다른 브라우저로 시도
